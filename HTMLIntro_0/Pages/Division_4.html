div, Division kelimesinin kısaltılmısı olarak HTML dünyasına giriş yapmıstır..Anlamı, tasarımsal alan demektir. Birbirinden bagımsız ve görsel acıdan cok daha iyi ve kontrol edilmesi kolay bir tasarım alanı sunar. Özellikle div tag'i hayatımıza girmeden önce tablolar yapılan tasarımlar hem performans acısından sıkıntılıydı hem de zor bir görsel tasarım işlemi yapmamıza sebep oluyordu...

Bazı temel Kavramlar

  • id => site üzerinde yer alan bir tag'i ifade etmek icin kullanılan attribute'tur..Bir Id benzersiz olmalıdır...Ancak zorlarsanız aynı Id'yi baska elementlere de verebilirsiniz.Tabii ki bu durum W3C kurallarına aykırıdır...Kullanılan bir Id aynı sayfadaki baska bir HTML elementi tarafından kullanılmamalıdır...
  • width => Bir elementin genişligini ifade eder...Bazı elementlerde tek basına bir attribute olarak yazılabilse de bazı elementlerde bu tanım CSS ister...
  • height => Bir elementin yüksekligini ifade eder...Bazı elementlerde tek basına bir attribute olarak yazılabilse de bazı elementlerde bu tanım CSS ister...
  • margin => left/top/bottom/right => Bir elementin ilgili taraftan dıs cerceveye olan uzaklıgını temsil eder
  • padding left/top/bottom/right => Bir elementin iceriginin ic cercevedeki sınıra olan uzaklıgını temsil eder
  • style - Bir css yazabileceginizi belirten attribute'tur...Biz bu tarzda yazılan css'e inline css deriz
